# LexBharat Architecture

## Vision
LexBharat is a law-first computing platform. The architecture is designed so that legal authority governs every layer of computation.

## Layered Architecture

```
User
  │
  ▼
Applications
  │
  ▼
AI Skills
  │
  ▼
Research & Workflow
  │
  ▼
Verified Knowledge
  │
  ▼
Primary Sources
```

## Core Principles

- Law is the authority.
- AI follows the law.
- Primary sources cannot be bypassed.
- Every conclusion must be explainable.
- Every legal assertion should be traceable to its source.

## Core Standards

- LRM – LexBharat Reference Model
- LAS – LexBharat Architecture Specification
- LKS – LexBharat Knowledge Standard
- LRS – LexBharat Reasoning Standard
- LXS – LexBharat Exchange Standard

These standards collectively define the constitutional layer of LexBharat OS.
