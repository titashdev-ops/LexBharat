# Source Authority Policy

## Purpose
This policy defines the authority hierarchy for all legal knowledge used within LexBharat.

## Source Levels

| Level | Authority | AI Reliance |
|------:|-----------|-------------|
| L1 | Constitution | Yes |
| L2 | Acts of Parliament | Yes |
| L3 | Rules & Regulations | Yes |
| L4 | Gazette Notifications | Yes |
| L5 | Supreme Court Judgments | Yes |
| L6 | High Court Judgments | Yes, with jurisdiction |
| L7 | Government Circulars | Context only |
| L8 | Books & Commentary | Background only |
| L9 | Blogs & AI Summaries | Never without verification |

## Core Principles
1. Primary sources always prevail.
2. Every legal statement should be traceable to an authoritative source.
3. AI must never invent citations.
4. Conflicts between sources are resolved according to the legal hierarchy.
5. All imported knowledge must include metadata and verification status.
